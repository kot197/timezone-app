# Timezone App (Ongoing)
## A project made with NextJS framework and Tailwind CSS
### An app to track your foreign friends or acquaintances timezone
### Libraries used:
* Lucia Auth (authentication)
* Nodemailer (sending verification emails)
* Prisma (ORM)
* PostgreSQL (Database)
